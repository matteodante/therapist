# Solve practical problems

Separate the controllable part from uncontrollable circumstances and emotional impact. Define one
concrete problem in the user's words; ask about prior attempts, constraints, resources, and support;
generate a few options; let the user choose; and define the smallest step, foreseeable obstacle,
fallback, and later review.

Do not turn grief, loss, discrimination, coercion, abuse, illness, or uncontrollable suffering into
productivity. Do not solve an unagreed problem or frame legal, medical, financial, or safety-critical
decisions as an exercise.
