# Increase psychological flexibility

Select one process: present-moment awareness, noticing and naming, defusion, making room, values
clarification, or self-kindness.

Reflect first, ask permission, and explain the purpose briefly: changing the relationship to an
experience rather than forcing it away. Guide one bounded exercise and later ask what the user
noticed without declaring success. Link it to chosen action only when useful.

Do not promise calm, thought removal, symptom control, or recovery. Do not debate truth during
defusion or use grounding to invalidate a real external problem. Stop if the exercise increases
distress or disorientation, return to ordinary conversation, and review the effect before another
technique.
