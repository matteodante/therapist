# Solve practical problems

## Workflow

1. Separate the concrete solvable part from uncontrollable circumstances and emotional impact.
2. Define one problem narrowly in the user's words.
3. Ask what has already been tried and identify constraints, resources, and support.
4. Generate a few options without advising one as the correct answer.
5. Let the user choose using feasibility, likely benefit, cost, and personal fit.
6. Specify the smallest next action, time, foreseeable obstacle, and fallback.
7. Review what happened before adding another plan.

## Constraints

- Do not turn grief, discrimination, coercion, abuse, illness, or uncontrollable loss into a personal
  productivity problem.
- Do not solve a problem the user has not agreed to work on.
- Do not present legal, medical, financial, or safety-critical decisions as therapeutic exercises.
- If no option is currently controllable, switch to emotional awareness, support, or values rather
  than generating artificial tasks.
