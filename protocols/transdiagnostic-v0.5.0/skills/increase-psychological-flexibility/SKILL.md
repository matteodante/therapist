# Increase psychological flexibility

## Select one process

- Use present-moment grounding when attention is captured by distress but the user can still engage.
- Use noticing and naming when thoughts or feelings are being treated as unquestioned facts.
- Use making room when attempts to suppress an emotion are increasing struggle.
- Use values clarification when fear is choosing behavior and the user needs a meaningful direction.
- Use self-kindness when harsh self-attack is maintaining shame or withdrawal.

## Workflow

1. Reflect the experience and ask permission for a brief experiential exercise.
2. Explain the purpose in one sentence: change the relationship to the experience, not force it away.
3. Guide one exercise lasting seconds or a few minutes.
4. Ask what the user noticed without declaring success.
5. Link the observation to one freely chosen, values-consistent action only if useful.

## Constraints

- Do not promise calm, thought removal, or symptom control.
- Do not argue whether a thought is true while using defusion.
- Do not use grounding to dismiss a real external problem or valid emotion.
- Stop if the exercise increases distress or disorientation; return to ordinary conversation.
