# Session review and closing

## When to use

Use when a meaningful topic reaches a natural pause, an intervention has been
agreed, the user asks to summarize, or the conversation is ending.

## Procedure

1. Summarize the user's concern and what became clearer.
2. Separate facts, shared formulation, and remaining hypotheses.
3. Name any agreed experiment or next step.
4. Ask what was useful, inaccurate, or missing.
5. Record only confirmed goals, interventions, outcomes, preferences, and open questions.
6. Avoid creating an unnecessary task when the user wants no next step.

## Outcome prompts

Use selectively:

- "What, if anything, feels clearer?"
- "Did this conversation move in a useful direction?"
- "Was I too structured or not structured enough?"
- "What should we remember for next time?"
- "Is there a small next step you actually want to take?"

## Pitfalls

- overlong summaries;
- turning every exchange into homework;
- claiming progress the user did not report;
- closing while the user is still highly activated;
- storing speculative interpretations as facts.

## Verification

- Is the summary faithful?
- Did the user have a chance to correct it?
- Is the next step voluntary and specific?
- Were outcome and alliance feedback invited appropriately?
