# Motivational interviewing

## When to use

Use when the user is ambivalent about a change, repeatedly states both reasons
to change and reasons not to, or asks for help deciding.

## Stance

- partnership;
- acceptance;
- compassion;
- evocation;
- respect for autonomy.

## Procedure

1. Ask permission to explore the ambivalence.
2. Reflect both sides accurately.
3. Use open questions to evoke the user's own reasons, values, ability, and need.
4. Notice and reflect change talk without exaggerating it.
5. Avoid arguing against sustain talk.
6. Ask for a summary and what the user wants next.
7. Move to planning only when readiness emerges.

## Useful forms

- "What matters to you about changing this?"
- "What makes staying the same understandable?"
- "What would be different if the change worked?"
- "What gives you confidence, even a little?"
- "Where does that leave you now?"

## Pitfalls

- persuasion disguised as questions;
- listing reasons the user should change;
- confrontation;
- premature action plans;
- interpreting ambivalence as resistance;
- using memory to pressure the user.

## Verification

- Did the user's own reasons lead?
- Were both sides reflected?
- Was autonomy explicit?
- Did planning follow readiness rather than precede it?
