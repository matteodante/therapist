# Collaborative formulation

## When to use

Use when several episodes provide enough information to propose a meaningful
pattern, or when choosing an intervention requires understanding the maintaining
mechanism.

## Formulation frame

Consider:

- current problem;
- trigger or precipitating situation;
- meaning attributed to it;
- thoughts, emotions, and body sensations;
- protective behavior or coping response;
- immediate benefit;
- longer-term cost;
- predisposing context only when supported and useful;
- protective factors and exceptions;
- alternative explanations.

## Procedure

1. Retrieve only relevant history.
2. Identify the smallest supported cycle.
3. Generate at least one plausible alternative explanation.
4. State the formulation tentatively.
5. Use ordinary language, not diagnostic shorthand.
6. Ask whether the user recognizes it.
7. Invite correction and counterexamples.
8. Record it as a working hypothesis unless explicitly shared and supported.
9. Use the formulation to choose the next question or intervention.

## Example structure

```text
When X happens,
you may interpret it as Y,
which brings Z,
so you protect yourself by doing A.
A helps in the moment, but may later produce B.
Does that fit, or is another explanation closer?
```

## Pitfalls

- hidden or definitive personality profiles;
- childhood explanations without evidence;
- pathologizing protective strategies;
- ignoring social, cultural, material, or relational context;
- using memory quantity as proof;
- confirmation bias.

## Verification

- What evidence supports this?
- What evidence does not?
- What alternatives remain?
- Did the user recognize or correct it?
- Does it lead to a useful and proportionate next step?
